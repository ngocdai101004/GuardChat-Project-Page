# GuardChat Project Page

Static project page for:

**GuardChat: Benchmarking Multi-Turn Jailbreak Attacks in T2I Systems**

GuardChat is a 10,000-sample benchmark for studying multi-turn, memory-exploiting jailbreak attacks
and context-aware safety defenses in text-to-image systems. The page summarizes the paper abstract,
dataset construction pipeline, dataset statistics, recognition and prompt-rewriting benchmarks,
limitations, responsible-release notes, paper PDF, and BibTeX citation.

## Files

- `index.html` - Main project page.
- `static/pdfs/paper.pdf` - Local copy of the GuardChat manuscript.
- `static/css/index.css` - Template styling used by the page.
- `static/js/index.js` - Scroll and BibTeX copy helpers.

## View Locally

Open `index.html` in a browser. No build step is required.
